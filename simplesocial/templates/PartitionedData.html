Partition Data

After reading in the data from a csv file, we have to partition the data set into training and testing datasets. Generally, 80/20 rule is followed when partitioning the data. 80% of the data is considered as training data, while the rest is used for testing the model. X is the input data, while T contains the target valuesSome of the functions used in the partitioning the data are Train, Model, RMSE, TrainSGD.For more information about the functions, please visit my github profile. We partition the data in order to calculate the weights assigned to the data set which will help to make good predictions.
